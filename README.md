# PEVC Website

Ohio State University Private Equity & Venture Capital club website — featuring a full on-site recruiting pipeline, project management portal, and admin CMS.

**Live site:** https://pevc.vercel.app/

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- React Router
- React Hook Form + Zod

## Features

- **Recruiting pipeline** — R0 application (KYC, resume, essays), R1/R2 interview scheduling with Calendly embed, case material delivery, and prep resources
- **Application tracker** — applicants see their real-time stage, deadlines, and hear-back dates
- **Admin CMS** — manage recruiting config (deadlines, essay prompts, case materials, resource links), kanban pipeline view, and full application detail drawer
- **Project portal** — PM/member task management, per-deliverable file uploads, client view
- **Admin content management** — team members, portfolio companies, partnership projects

## Local Development

```sh
git clone https://github.com/yuvi-atre/PEVC-website
cd PEVC-website
npm install
npm run dev
```
